2.Java类和对象

属性名称;
返回值类型 方法名称(){}
}

操作要用到对象，对象定义如下：

类名称 对象名称 = new 类名称() ;

访问类中方法或属性：

访问类中的属性：对象.属性 ;
调用类中的方法：对象.方法();

3.封装性

3.1 封装性的概念

封装性是面向对象思想的三大特征之一，封装就是隐藏实现细节，仅对外提供访问接口。实现细节部份包装、隐藏起来的方法。

封装有：属性的封装、方法的封装、类的封装、组件的封装、模块化封装、系统级封装
封装的好处：模块化、信息隐藏、代码重用、插件化易于调试、具有安全性
封装的缺点：**会影响执行效率
